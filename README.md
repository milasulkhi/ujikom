\# Uji Kompetensi



Project ini merupakan aplikasi sederhana untuk memenuhi tugas \*\*Uji Kompetensi (Ujikom)\*\* dengan fitur login dan dashboard menggunakan PHP \& MySQL.



\---



\## Fitur

\- Login user (email \& password)

\- Dashboard

\- CRUD data (Tambah, Tampil, Edit, Hapus)

\- Logout



\---



\## Teknologi

\- PHP Native

\- MySQL

\- HTML, CSS

\- Bootstrap
